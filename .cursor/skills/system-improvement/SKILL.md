# 시스템 개선 스킬

에이전트 시스템 성능 분석과 자동 최적화를 위한 전문 스킬입니다.

## 사용 시점

- 에이전트 성능 메트릭을 분석할 때
- 워크플로우 병목을 탐지할 때
- 규칙 변경 A/B 테스트를 수행할 때
- 에이전트 규칙과 스킬을 최적화할 때
- 성능 리포트를 생성할 때
- 시스템 상태 추세를 모니터링할 때

## 트리거 조건

- 성능 저하 10% 이상
- 연속 3회 이상 실패
- 병목 중요도 임계값 초과
- 주간 요약 생성 시점

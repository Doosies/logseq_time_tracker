# 에이전트 선택

## 선택 규칙

### 기획 에이전트
**호출 조건**:
- 요구사항 분석 필요
- 아키텍처 설계 필요
- API 설계 필요

### 구현 에이전트
**호출 조건**:
- 코드 작성
- 리팩토링
- 버그 수정

### QA 에이전트
**호출 조건**:
- 테스트 작성
- 코드 리뷰
- 커버리지 체크
- 버그 재현

### 문서화 에이전트
**호출 조건**:
- JSDoc 작성
- README 업데이트
- CHANGELOG 작성

## 서브태스크 매칭

```markdown
서브태스크 유형별 에이전트 조합:

설계 → 기획
구현 → 구현
테스트 → QA
문서 → 문서화
전체 기능 → 기획 + 구현 + QA + 문서화
```

- [ ] 에이전트 조합 최적화

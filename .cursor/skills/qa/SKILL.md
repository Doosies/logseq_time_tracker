# QA Skill

Specialized skill for testing and quality assurance.

## When to Use

- Writing unit/integration/E2E tests
- Code review
- Coverage measurement (target 80%+)
- Storybook story verification
- Performance regression testing
- Svelte component testing

## Quality Gates

- Test pass rate: 100%
- Coverage target: 80%+
- Performance regression: within 10%
- Security vulnerabilities: 0
- All stories must have play functions

# 기획 스킬

요구사항 분석과 아키텍처 설계를 위한 전문 스킬입니다.

## 사용 시점

- 새 기능 요구사항을 분석할 때
- 시스템 아키텍처를 설계할 때
- RESTful API를 설계할 때
- 기술 스택을 선정하고 근거를 정리할 때
- 데이터 모델을 설계할 때

## 핵심 원칙

- 리소스 중심 URL 설계(동사보다 명사)
- 기존 시스템과 일관된 아키텍처 유지
- 제약사항과 트레이드오프 문서화
- 테스트 가능한 구조로 설계

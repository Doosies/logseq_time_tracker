# Planner Skill

Specialized skill for requirements analysis and architecture design.

## When to Use

- Analyzing user requirements for new features
- Designing system architecture
- RESTful API design
- Tech stack selection and justification
- Data model design

## Key Principles

- Resource-oriented URL design (nouns, not verbs)
- Consistent architecture with existing system
- Document all constraints and trade-offs
- Design for testability

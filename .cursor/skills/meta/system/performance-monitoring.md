# 성능 모니터링

## 수집 데이터

### 사이클 레벨
- 전체 실행 시간
- 토큰 사용량  
- 재시도 횟수
- 성공/실패

### 에이전트 레벨
- 에이전트별 실행 시간
- 에이전트별 재시도
- 품질 점수

## 분석 지표

### 효율성
- 평균 완료 시간
- 토큰 효율 (토큰/태스크)
- 에이전트 호출 횟수

### 품질
- 에러율
- 재시도 횟수
- 테스트 통과율

### 추세
- 시간에 따른 변화
- 개선 효과 측정

## 분석 트리거

- 20개 사이클마다 자동
- 에러율 30% 초과 시
- 평균 시간 50% 증가 시

- [ ] 분석 지표 계산

### 원인 1: 원칙 인지 부족
- **현상**: "메인 에이전트 금지" 원칙을 무시하고 코드 직접 수정
- **원인**: main-orchestrator.mdc에 체크포인트가 명확하지 않음
- **증거**: 메인 에이전트가 StrReplace 도구로 직접 UserScriptSection.svelte 수정

### 원인 2: 워크플로우 참조 누락
- **현상**: `.cursor/workflows/plan-execution-workflow.md` 읽지 않고 진행
- **원인**: 워크플로우 문서를 참조하도록 강제하는 메커니즘 부재
- **증거**: 0단계와 9단계를 건너뜀

### 원인 3: 체크리스트 부재
- **현상**: 각 단계별 완료 조건 확인 없음
- **원인**: 단계별 체크포인트가 문서화되어 있지 않음
- **증거**: 서브에이전트 호출 여부를 확인하지 않고 직접 수행

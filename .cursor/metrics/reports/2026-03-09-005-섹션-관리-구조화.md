## 4. 주요 결정사항

| 단계 | 결정 | 근거 | 검토한 대안 |
|------|------|------|-------------|
| developer | `<section.component />` dot notation 사용 | Svelte 5 Runes 모드에서 `svelte:component` deprecated | svelte:component 유지 (경고 발생) |
| developer | `{@const}` snippet/#each 직접 자식 배치 | Svelte 규칙: @const는 snippet, #if, #each 등 직접 자식만 허용 | div 내부 배치 (컴파일 오류) |
| developer | `#sections` 경로 alias 추가 | 단일 모듈 import로 레지스트리와 유틸 사용 | 상대 경로 import |

## 5. 발견된 이슈 및 해결

| 단계 | 이슈 | 해결 방법 | 영향도 |
|------|------|-----------|--------|
| developer | `{@const}` 유효 부모 제한 | `{@const}`를 `{#snippet}` 또는 `{#each}` 직접 자식으로 이동 | none |
| developer | 기존 storage에 'calculator' 존재 | mergeWithDefault가 알 수 없는 ID 필터링, user-script 추가 | minor |

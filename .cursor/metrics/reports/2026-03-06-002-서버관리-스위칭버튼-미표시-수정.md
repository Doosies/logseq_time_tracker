## 1.7 주요 결정사항

| 단계 | 결정 | 근거 | 검토한 대안 |
|------|------|------|-------------|
| implementation | `flexShrink: 0` 추가 | flex 기본값 1이라 `flex: 1` 형제에 밀려 축소됨 | `flex: 0 0 auto` (동일 효과이나 속성 1개가 더 간결) |
| implementation | SVG 아이콘으로 교체 | `⇄` 유니코드가 시스템 폰트에서 미렌더링 | 다른 유니코드(`↔`, `⇔`) 시도 (폰트 의존성 동일) |
| implementation | 버튼 스타일 명시적 리셋 | 브라우저 기본 버튼 스타일(3D 테두리)이 이상하게 보임 | 글로벌 CSS 리셋 (영향 범위 넓음) |
| qa | 테스트 미작성 | CSS/시각적 변경만, 회귀 위험 낮음, 비즈니스 로직 무관 | 최소 1개 테스트 (비용 대비 효과 낮음) |

# Introduction

Personal Logseq Plugin은 React와 TypeScript로 작성된 현대적인 Logseq 플러그인입니다.

## 특징

- **🚀 빠른 개발**: Vite 7.3 기반의 초고속 HMR
- **📦 모노레포**: Turborepo와 pnpm 워크스페이스
- **⚛️ React 19**: 최신 React 기능 활용
- **🎯 TypeScript**: 완벽한 타입 안전성
- **🧪 Vitest**: 빠른 테스트 실행
- **🎨 코드 품질**: ESLint + Prettier

## 왜 이 플러그인을 사용해야 하나요?

이 플러그인 템플릿은 Logseq 플러그인 개발을 위한 최신 도구와 모범 사례를 제공합니다. 빠른 개발, 테스트, 배포를 위한 완벽한 환경을 갖추고 있습니다.

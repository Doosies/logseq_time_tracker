# Introduction

Personal 모노레포는 Svelte 5와 TypeScript 기반의 개발 도구 모음입니다.

## 패키지

- **@personal/uikit** - Svelte 5 기반 공유 UI 컴포넌트 라이브러리 (vanilla-extract 스타일링)
- **@personal/ecount-dev-tool** - 이카운트 개발 환경 관리 Chrome 확장프로그램
- **@personal/mcp-server** - Cursor용 MCP (Model Context Protocol) 서버
- **@personal/time-tracker** - Logseq 시간 추적 플러그인
- **@personal/docs** - VitePress 문서 사이트 (현재 보고 있는 사이트)

## 특징

- **Svelte 5 Runes**: 최신 Svelte 반응성 시스템
- **TypeScript**: 전 패키지 타입 안전성
- **Turborepo**: 모노레포 빌드 캐싱 및 병렬 실행
- **pnpm Workspace**: 효율적인 의존성 관리
- **Storybook 10**: UI 컴포넌트 시각적 테스트 및 문서화
- **Vitest**: 빠른 단위/통합 테스트

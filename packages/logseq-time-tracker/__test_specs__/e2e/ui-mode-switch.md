# UI 모드 전환 E2E 테스트 스펙

> **패키지**: `@personal/logseq-time-tracker`
> **테스트 레벨**: e2e
> **ID 체계**: `UC-E2E-{번호}` (3자리 zero-padded)

---

#### UC-E2E-006: 전체 화면 열기 시 툴바는 숨고 타이머 영역과 돌아가기가 보인다
- **Given**: 앱이 로드되어 툴바가 표시되어 있다
- **When**: "전체 화면 열기" 버튼을 클릭한다
- **Then**: 툴바는 숨고, 타이머 섹션과 "돌아가기" 버튼이 보인다

#### UC-E2E-007: ESC로 숨긴 뒤 모의 툴바로 다시 열면 전체 화면 상태가 유지된다
- **Given**: 전체 화면 상태이다
- **When**: ESC로 UI를 숨긴 뒤 모의 툴바 버튼으로 재열기한다
- **Then**: 전체 화면 상태가 유지된다

#### UC-E2E-008: 돌아가기로 작은 툴바로 전환하면 타이머 섹션은 숨는다
- **Given**: 전체 화면 상태이다
- **When**: "돌아가기" 버튼을 클릭한다
- **Then**: 툴바가 보이고 타이머 섹션은 숨는다

#### UC-E2E-009: 작은 툴바 상태에서 ESC 후 재열기 시 툴바 상태가 유지된다
- **Given**: 돌아가기로 작은 툴바 상태이다
- **When**: ESC로 UI를 숨긴 뒤 모의 툴바 버튼으로 재열기한다
- **Then**: 작은 툴바 상태가 유지된다

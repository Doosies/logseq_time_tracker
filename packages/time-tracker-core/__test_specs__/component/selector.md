# 셀렉터 테스트 스펙

> **패키지**: `@personal/time-tracker-core`
> **테스트 레벨**: component
> **ID 체계**: `UC-{영역}-{번호}` (3자리 zero-padded)
> **형식**: BDD (Given-When-Then)

---

### 4.4 셀렉터

#### UC-UI-009: 폴더 중첩 탐색

- **Given**: "프로젝트 > 메인업무 > 작업" 구조의 Category 트리가 있다
- **When**: "프로젝트"를 클릭 → "메인업무"를 클릭한다
- **Then**: "작업" 레벨의 항목들이 표시된다
- **Phase**: 3
- **테스트 레벨**: 컴포넌트

#### UC-UI-010: 셀렉터 검색

- **Given**: 10개 이상의 Category 항목이 있다
- **When**: 검색창에 "개발"을 입력한다
- **Then**: "개발"을 포함하는 항목만 필터되어 표시된다
- **Phase**: 3
- **테스트 레벨**: 컴포넌트
